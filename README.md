# Day 1: Basic Logic Gates Design and Verification

## Overview
This project implements the foundational digital logic gates (AND, OR, NOT, XOR) in Verilog HDL using **Dataflow Modeling** (`assign` statements). The design has been fully verified using a comprehensive testbench on **ModelSim**.

## Hardware Specifications
- **Inputs:** 1-bit `a`, 1-bit `b`
- **Outputs:** `out_and`, `out_or`, `out_not1`, `out_not2`, `out_xor`

## Tools Used
- **Language:** Verilog HDL
- **Simulation/Verification Tool:** Mentor Graphics ModelSim

## Key Learnings
- Understood the concept of continuous assignment using the `assign` keyword.
- Learned how to map behavioral truth tables directly into gate-level simulation vectors.
- Fixed common syntax errors regarding procedural block compilation and port instantiation in testbenches.
